# 167. |阶乘

## Question description


请编写一个程序，从键盘上输入n（n 的范围是1～20），求n 的阶乘。


## Solution

Language: **['Java']**

```
import java.util.Scanner;

public class Main {


		public static void main(String[] args) {
		        Scanner in = new Scanner(System.in);
		        int m=in.nextInt();
		        for(int i=0;i<m;i++){
		        	int n=in.nextInt();
		        	long sum=1;
		        	for (int j = 2; j <=n; j++) {
						sum*=j;
					}
			        System.out.println(sum);
		        }
		        
		    }
		
}

```
